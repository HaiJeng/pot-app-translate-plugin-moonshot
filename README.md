# pot-app-translate-plugin-moonshot

月之暗面翻译插件 - 一个为pot-app开发的翻译插件

## 简介

这是一个基于月之暗面(Moonshot) API的翻译插件，支持多语言翻译，提供高质量的翻译结果。

## 前置要求

- [pot-app](https://github.com/pot-app/pot-app) - 请先下载并安装pot-app
- [月之暗面开发者平台](https://platform.moonshot.cn/)- 需要在月之暗面官网申请API密钥

## 功能特点

- 支持多种月之暗面模型选择（v1-8k/v1-32k/v1-128k）
- 支持多种语言之间的互译
- 提供专业、流畅的翻译结果

## 安装方法

1. 在pot-app中点击"偏好设置"
2. 选择"服务设置"
3. 点击"翻译-添加外部插件"
4. 选择本插件的发布包进行安装

## 配置说明

1. 选择翻译模型（默认 moonshot-v1-8k）
2. 输入月之暗面 API密钥

## 使用说明

安装并配置完成后，在pot-app的翻译引擎列表中选择"moonshot"即可使用。

## 注意事项

- 请妥善保管你的API密钥
- 翻译服务需要联网使用
